üìã CSV Format Requirements

Your CSV file should contain the following columns in order:

  • Date Applied (required) - Date you applied for the job
  • Job Title (required) - Position title
  • Company (required) - Company name
  • Status (optional) - Application status
  • Job URL (optional) - Link to job posting
  • Notes (optional) - Additional notes

‚ö†Ô∏è Important Notes

  • First row can be headers (they will be automatically detected and skipped)
  • Date formats supported: YYYY-MM-DD, MM/DD/YYYY, DD/MM/YYYY, and more
  • If Status is empty, it will default to "Applied"
  • Empty rows will be skipped
  • Maximum file size: 10MB
